Descripción del taller

El taller de aplicaciones de una sola página (MVC) AngularJS esta diseñado para brindar al participante las mejores técnicas para crear aplicaciones bajo el modelo vista controlador.

Objetivos

Guiar a los alumnos a través de la construcción de una aplicación: a partir del diseño de la interfaz de usuario, incluyendo la escritura de la lógica del negocio, hasta las pruebas finales.

 Condiciones

En cuanto a los requisitos, costos, horarios y temas relacionados aplican todas las condiciones del Diplomado Webmaster con certificación de 3CT.

 

Durante mucho tiempo, se desarrollaron aplicaciones web para seguir un modelo de ida y vuelta. El navegador solicita un documento HTML inicial, del servidor. Las interacciones del usuario, como hacer clic en un enlace o enviar un formulario, hace que el navegador solicite y reciba un nuevo documento HTML. En este tipo de aplicación, el navegador es esencialmente un motor de navegación para el contenido HTML, y toda la lógica de la aplicación y los datos reside en el servidor. El navegador hace una serie de peticiones HTTP sin estado, que el servidor maneja mediante la generación de documentos HTML dinámicos.

Las aplicaciones de una sola página, adoptan un enfoque diferente. Un documento HTML inicial se envía al navegador, pero las interacciones del usuario conducen a peticiones Ajax para actualizar pequeños fragmentos de HTML, es decir, el documento HTML inicial no se vuelve a cargar.

La naturaleza declarativa de angularjs hace que sea más fácil escribir y entender las aplicaciones

Una aplicación AngularJS, requerirá menos líneas de código, para completar una tarea que una solución JavaScript pura usando jQuery.

Las plantillas de aplicación angularjs están escritas en HTML puro, por lo que los diseñadores les resultará más fácil trabajar con ellos.

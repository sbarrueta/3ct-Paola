Temas que se ven en el Diplomado.



Creación de Base de Datos

Diseño de Bases de Datos, aplicando ampliamente el modelo entidad-relación. Queries utilizando operadores comúnes

Backup/ Restore

Carga de información masiva, actualización y eliminación de registros.

Subconsultas avanzadas

Uso de consultas avanzadas y uniones complejas, así como la aplicación de vistas.

Trancac SQL Server

Creación de Store Procedures, funciones y triggers. Programación en la capa de Base de Datos.

Administración

Recuperación de información, implementación de seguridad, administración de acceso, creación de perfiles, aplicación de repaldos.

Introducción a Tuning y Optimización

Conocer la estructura de almacenamiento lógico y físico de SQL Server, particionar tablas, alinear índices, interpretar planes de ejecución, monitorear procesos, desfragmentar, detectar bloqueos

Analista SQL

Se cubren los conocimientos necesarios para extracción de información, creación de consultas y reportes, funciones y herramientas necesarias para el analisis de las bases de datos

Diseño de Bases de Datos

Crearas una base de datos desde cero, pasando por las mejores tecnicas y convenciones de desarrollo correcto de una base de datos relacional.

Administración

Si te interesa hacer la administración de una base de datos también en este diplomado aprenderas a crear permisos, dar alta de nuevos usuarios y respaldos de información necesarios para una buena administración de información

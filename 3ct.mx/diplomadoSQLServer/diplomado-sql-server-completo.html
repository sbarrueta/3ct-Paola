Diplomado SQL Server

Descripción del Diplomado

El Diplomado SQL Server está enfocado en brindar al participante los conocimientos, las estrategias y la práctica necesarios para desarrollarse en el área empresarial de las bases de datos. El aprendizajes inica desde los conceptos básicos hasta la práctica avanzada de administración y programación.

Objetivo

Aprender a diseñar, programar, administrar y optimizar bases de datos con MS SQL Server, ya sea desde cero o para dar mantenimiento a bases de datos ya en producción, de manera que el participante pueda apoyar cualquiera de las 4 áreas involucradas : diseñar, programar, administrar u optimizar bases de datos.

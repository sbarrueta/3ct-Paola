Introducción a Dreamweaver básico: diseño web con la paquetería Adobe

Dreamweaver es el editor más moderno de sitios web (la diferencia entre sitio y página se basa en su tamaño; es decir, página es un documento HTML y sitio es un proyecto grande que puede contener incluso miles de documentos en distintos formatos). Además, Dreamweaver es una herramienta muy solicitada en el mercado laboral, de hecho, actualmente, el 90% de los desarrolladores HTML profesionales lo usan; su principal ventaja es la flexibilidad para organizar y vincular conjuntos grandes de archivos y conformar sitios avanzados. La aplicación está integrada por numerosas funciones que facilitan el proceso de diseño de un proyecto. Tiene, entre muchas otras cosas, validadores de código, verificadores de hipervínculos, un cliente FTP integrado (que permite la publicación en un servidor), etc.

Dreamweaver sirve para dar diseño a las páginas estáticas así como a las dinámicas. Admite el desarrollo de código fuente en varios formatos, tales como tecnologías de servidor y lenguajes de programación: XHTML, CSS, JavaScript, XML, PHP, JSP, ASP, Coldfusion, etc.

Aunque el objetivo final de Dreamweaver es crear código, se puede utilizar la interface gráfica para aligerar el proceso de desarrollo, dándole rapidez y facilidad; especialmente para las personas sin conocimientos profundos en el campo de la programación, tales como diseñadores gráficos.

De entre todas las herramientas de su tipo, Adobe Dreamweaver destaca por sus numerosas e ingeniosas técnicas para desarrollar y actualizar el código fuente, y está disponible para plataformas Windows y Mac. Actualmente, en 3CT se está usando la versión más reciente de la paquetería web: el poderoso Dreamweaver CS5.

El curso de Dreamweaver básico es para personas sin previo conocimiento en la creación de páginas web, que deseen acceder a este proceso desde el principio hasta nivel intermedio. Es recomendable que antes de empezar el curso de Dreamweaver tengan un conocimiento suficiente del uso de algún programa de edición de gráficos; de preferencia, de la paquetería Adobe Web (por ejemplo Photoshop, Fireworks, Flash, etc.).

 

Objetivos del curso

Una vez concluido el curso, el participante podrá, entre muchas otras cosas:

  • Entender los principios de funcionamiento de la Internet actual y del diseño web
  • Crear páginas (X)HTML con texto, imágenes, multimedia, hipervínculos, etc.
  • Estructurar la disposición de la información con tablas y marcos
  • Enviar sitios al servidor y actualizarlo mediante el cliente FTP integrado a Dreamweaver

Al terminar el curso de Dreamweaver básico, el alumno puede, automáticamente (sin examen necesario), inscribirse al curso de Dreamweaver intermedio donde aprenderá a trabajar con XHTML, CSS y Javascript.

Ejercicios

El curso de Dreamweaver se basa en ejercicios prácticos asignados por el instructor, o algunos propuestos por los mismos participantes del grupo. De esta manera, después del desarrollo del temario, el alumno podrá, entre otras cosas:

  • Crear proyectos con varios sistemas de navegación
  • Crear un formulario interactivo a la medida de sus necesidades
  • Abrir una cuenta en un servidor comercial o en uno gratuito y subir un sitio en línea

Temario

El curso de Dreamweaver está pensado para iniciar desde el punto de vista más elemental ya que está dirigido a participantes que desconocen esta herramienta de diseño. Por este motivo, el temario del curso de Dreamweaver incluye conceptos útiles que se han estructurado de la siguiente manera:

  • Introducción al ambiente y el diseño de sitios web
  • Planeación y definición de sitios web
  • Estructura y creación de documentos (X)HTML
  • Creación y actualización de hipervínculos
  • Uso de botones tipo rollover, botones Flash, mapas de imágenes y anclajes.
  • Propiedades del documento HTML
  • Texto y tipografía en el diseño web
  • Caracteres especiales
  • Listas ordenadas, sin orden y de definición.
  • Introducción a estilos CSS integrados en la cabecera del documento HTML
  • Imágenes y multimedia considerando los formatos permitidos y la posible optimización usando Photoshop
  • Creación de galerías de imágenes web
  • Tablas y creación de páginas splash
  • Marcos (Frameset) y barras de navegación
  • Diseño de formularios
  • Configuración del cliente FTP integrado y sincronización del contenido entre los archivos locales y el sitio remoto
 

Ventajas adquiridas

No sólo basta saber operar la herramienta, es por eso que nuestro curso integra los temas necesarios para elevar el aprovechamiento de Dreamweaver. Al terminar el curso, el alumno podrá:

  • Entender los principios y las reglas de construcción de documentos (X)HTML para la validación correcta de los códigos y el posicionamiento de su proyecto en los motores de búsqueda (SEO)
  • Aplicar los principios de uso correcto de la tipografía y el color en el entorno de la Internet
  • Distinguir entre las propiedades de varios formatos gráficos y de multimedia aceptados por los navegadores modernos
  • Iniciarse en los conceptos de tecnologías que forman la programación web moderna, tales como CSS y JavaScript

Preguntas frecuentes

¿Si aprendo el curso de Dreamweaver, necesito saber hacer el código HTML manualmente?
En teoría, no es necesario; no obstante, es altamente recomendado, para trabajar profesionalmente, aprender lo mismo HTML que CSS
¿Es cierto que Dreamweaver puede crear elementos de tecnología AJAX como menús desplegables y acordeones?
Sí, aunque ese tema no se ofrece en este curso sino en el de Dreamweaver intermedio
¿Dreamweaver usa otras aplicaciones?
Sí, el alumno que ha cursado antes un curso de Photoshop o de Flash puede aprovechar mucho más la integración de varias tecnologías de la paquetería Adobe.

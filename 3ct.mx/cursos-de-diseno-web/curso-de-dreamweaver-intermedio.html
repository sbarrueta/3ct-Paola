Curso de Dreamweaver intermedio

El curso de Dreamweaver intermedio se enfoca en enseñar las técnicas avanzadas para crear y mantener sitios web grandes, con énfasis en hojas de estilo externas (CSS), la interactividad de AJAX y las técnicas de aumento de la productividad.

Introducción a Dreamweaver intermedio: diseño web con la paquetería Adobe

El diseño web a principios del siglo XXI es muy diferente al de hace una década, cuando un desarrollador web podía trabajar con herramientas más sencillas tipo bloc de notas y sacar proyectos exitosos. Ahora, con la cantidad de datos de varios formatos que componen un sitio (por ejemplo texto extenso, múltiples imágenes en varios formatos, fotografías, multimedia, aplicaciones que corren en servidor, bases de datos y demás), es casi imposible crear y dar mantenimiento a proyectos extensos que requieren cambios frecuentes, sin recurrir a tecnologías y herramientas adecuadas y suficientemente poderosas. Dreamweaver descuella como una aplicación sumamente sofisticada que proporciona tanto a los diseñadores web como a los desarrolladores un excelente y altamente productivo flujo de trabajo.

Es una herramienta completa que facilita la creación de sitios de todos tamaños desde el concepto y los bocetos iniciales, mediante creación de los códigos XHTML, CSS y JavaScript hasta el desarrollo de aplicaciones de servidor con PHP o JSP y su conexión con bases de datos. La fácil integración de Dreamweaver con otras aplicaciones de la paquetería Adobe, como Photoshop o Flash es otra ventaja que es causa de su popularidad entre los webmasters.

En el curso de Dreamweaver Intermedio el temario se enfoca en la realización de sitios grandes, su posterior mantenimiento, administración y actualización. Se introduce hojas de estilos externos CSS que permiten asegurar la consistencia visual de varios elementos de las páginas del sitio. Con técnicas de productividad como las bibliotecas y las plantillas, los usuarios pueden controlar y mantener el contenido de todos los documentos de su sitio. Además, se introducen técnicas muy populares relacionados con la tecnología Ajax, tales como bibliotecas Spry integradas con Dreamweaver y algunas externas que permiten realizar efectos tipo Light box, etc.

El curso de Dreamweaver intermedio requiere que el participante tenga ya conocimientos básicos de las tecnologías que se utiliza en el curso, es indispensable tener dominado el temario del curso de Dreamweaver básico (alternativamente sería posible tener alto nivel de HTML) y de Photoshop (o algún otro editor grafico de mapas de bits). Al terminar el curso de Dreamweaver intermedio en 3CT, el alumno se califica de manera automática (sin examen) para ser admitido Diplomado webmaster con certificación de 3CT y al Taller integral de diseño web.

 

Objetivos del curso

Una vez concluido el curso, el participante podrá, entre muchas otras cosas:

  • Estructurar el contenido con XHTML, darle la presentación con CSS e incluir interactividad de JavaScript
  • Crear y administrar sitios web grandes y compuestos de una variedad de componentes
  • Crear, conseguir y modificar para ajustar a sus necesidades las plantillas CSS
  • Validar los códigos y hacer otros tipos de revisiones automatizadas de sus proyectos

Al terminar el curso de Dreamweaver intermedio, el alumno puede, automáticamente (sin examen necesario), inscribirse al Taller integral de diseño web donde puede desarrollar un proyecto personal o profesional de su preferencia.

Ejercicios

El curso de Dreamweaver intermedio contempla introducir los conceptos a los alumnos de manera teórica y demostrativa ejecutada por el instructor y luego reafirmar los conocimientos adquiridos a través de una serie de ejercicios prácticos:

  • Crear diferentes presentaciones de un sitio web conectándolo a varias hojas de estilo
  • Desarrollar un mapa interactivo con Ajax
  • Integrar un menú de varias páginas del sitio cuyas instancias se actualizaran al modificar el objeto maestro

Temario

El curso de Dreamweaver intermedio está pensado para ofrecer al alumno nuevas técnicas de productividad que mejoraran y ampliaran sus conocimientos adquiridos en el curso de Dreamweaver básico. Se puede considerar como su continuación natural y su temario tiene la siguiente estructura:

  • Introducción al modelo XHTML + CSS + JavaScript
  • Creación y vinculación de hojas de estilo externas CSS
  • Las más indispensables propiedades de CSS
  • Comportamientos de JavaScript
  • Creación de estructura mediante los elementos DIV y SPAN
  • Creación de menús desplegables con CSS y AJAX
  • Creación de los elementos arrastrables
  • Validación de formularios con JavaScript
  • Validación de formularios con Ajax
  • Anidación de elementos interactivos que se cambian en el navegador del usuario
  • Integración de un elemento IFRAME (marco flotante)
  • Integración de las bibliotecas de Dreamweaver
  • Creación y administración de las plantillas
  • Revisiones automatizadas del sitio (Resultados)
  • Validación de los códigos XHTML y CSS
  • Utilizar Dreamweaver como herramienta para desarrollar los códigos a mano
 

Ventajas adquiridas

El curso de Dreamweaver intermedio no es una serie de simples técnicas enseñadas una tras otra, es la enseñanza más compleja que embarca también otros ámbitos no relacionados directamente con el mero uso de la aplicación sino intentando instruir cómo usarla bien y para el mejor beneficio del usuario final. Al terminar el curso, el alumno va a saber:

  • Algunas de las características del más moderno diseño grafico para web y su realización con Photoshop y Dreamweaver
  • Algunos conceptos de tecnologías que todavía no forman el estándar contemporáneo y no están incluidas en Dreamweaver pero se pueden utilizar en ciertas situaciones, por ejemplo algunas propiedades de CS3
  • Distinguir entre varias tecnologías, sus ventajas y puntos débiles y recomendaciones, cuándo conviene usar cada una de ellos (por ejemplo el dilema de Flash vs. Ajax)
  • Sacar provecho de varios sitios y herramientas web que ofrecen numerosas utilerías y recursos indispensables para diseñadores y desarrolladores web

Preguntas frecuentes

¿El curso de Dreamweaver Intermedio incluye enseñanza de PHP y conexiones con bases de datos?
No. El temario es estrictamente enfocado en el desarrollo front-end. Si deseas conocer PHP y MySQL, puedes estudiar cursos de programación y bases de datos en 3CT. Al saber estas técnicas y Dreamweaver no vas a tener mayor problema de integrar tus conocimientos y realizar conexiones con bases de datos desde Dreamweaver, o uno de tus instructores te podrá ayudar.
¿Que son las plantillas que menciona el temario? ¿He oído que no es muy profesional usar plantillas?
Hay dos tipos de plantillas (templates) que vemos en este curso. Las plantillas de Dreamweaver (Dreamweaver Templates o DWT) son paginas tipo machote creadas por ti en las que puedes basar el diseño de tu sitio, al modificar la pagina maestra, todas las otras páginas del sitio se actualizaran para reflejar los cambios. También existen plantillas gratuitas de CSS libres para descargar y si es cierto que es algo mal visto usarlas si se trata de servicios y sitios profesionales. Pero conociendo las técnicas enseñadas en el curso de Dreamweaver intermedio puedes usarlas como un punto de partida y ajustar su aspecto a tus necesidades sin depender de soluciones graficas que utilizo el autor de la plantilla.
¿Cuanto de Ajax se ve en este curso?
El temario del curso de Dreamweaver intermedio contempla la enseñanza de cómo usar las bibliotecas Spry que es una aplicación de la tecnología Ajax de Adobe y algunas demostraciones de otras bibliotecas. Si te interesa precisamente este tema, recomendamos estudiar el curso de Dreamweaver intermedio para obtener las nociones generales sobre que es Ajax y luego profundizar tus conocimientos en el segundo Taller del Diplomado webmaster con certificación de 3CT. El Taller de desarrollo de interfaces dinámicas es contemplado precisamente para los egresados del curso de Dreamweaver intermedio, sobre todo los que cuenta con el perfil de diseñador web.

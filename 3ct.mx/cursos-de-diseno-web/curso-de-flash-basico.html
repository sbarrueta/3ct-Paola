Curso de Adobe Flash básico

Adobe Flash es una de las herramientas más populares para crear contenidos interactivos empaquetados en presentaciones que se pueden ejecutar, virtualmente, desde cualquier plataforma. Su uso más popular es el desarrollo de animaciones aunque pueden crearse programas con interface gráfica. Los archivos de estas presentaciones pueden incrustarse en una página web y ser ejecutados desde el navegador; asimismo, pueden ser archivos que corren directamente desde el sistema operativo.

El curso de Adobe Flash está planteado para personas interesadas en el diseño gráfico digital que tengan conocimientos básicos de algún editor profesional de imágenes como Photoshop o Illustrator. La interacción que tiene Adobe Flash con las páginas web es tan importante que el curso también está orientado para quienes desean enriquecer los sitios web con contenidos interactivos o animados.

Introducción a Adobe Flash

Flash es un programa que sirve para la creación de presentaciones multimedia. Utiliza una serie de herramientas para crear animaciones de modo gráfico vectorial, e igualmente permite integrar gráficos de mapas de bits, sonidos, películas videos, etc. Una de las ventajas más importantes de Flash es que puede trabajar como interface hacia una base de datos, utilizando páginas dinámicas como PHP, MySQL, XML, etc.; comunicándolas con XHTML a través del uso de JavaScript.

Además, Flash utiliza el lenguaje de programación llamado ActionScript con el que se puede agregar interactividad a las presentaciones o crear video-juegos avanzados en cualquier dispositivo portátil (Hand Held, teléfono celular, etc.). Aunque igualmente se pueden hacer imágenes con un alto nivel de animación que pueden servir para enriquecer la página principal de un sitio web. Esta posibilidad de mejorar la apariencia de los contenidos de la Internet hizo que Adobe Flash obtuviera una gran popularidad.

Se pueden hacer aplicaciones que se reproduzcan directamente del sistema operativo en las que haya una simple sucesión interactiva de imágenes con contenido multimedia (como un tutorial o un examen) hasta aplicaciones completamente orientadas a la lógica de programación y revestidas con las herramientas gráficas que Flash ofrece.

Con Flash se puede acceder a uno de los ambientes más exitosos del ambiente gráfico: el desarrollo de video juegos. De hecho, las páginas que ofrecen la oportunidad de jugar en línea muy frecuentemente presentan los recursos con archivos incrustados interactivos hechos con Flash.

Para que las animaciones creadas con lo aprendido en el curso de Adobe Flash básico funcione en los navegadores web hace falta descargar e instalar el Adobe Flash Player que permite visualizar archivos SWF que son los que se producen con la herramienta en cuestión, es decir aplicaciones que pueden tener lo mismo imagen que sonido o contenidos de texto.

La importancia de Flash se alcanza justamente por el contenido multimedia: utiliza gráficos vectoriales e imágenes rasterizadas, así como sonido, código de programa, flujo de vídeo y audio. La posibilidad de combinar todo esto le da a las aplicaciones una profunda versatilidad y es tan flexible que se puede ejecutar virtualmente en cualquier plataforma.

 

Objetivos del curso

Al terminar el curso el alumno será capaz de:

  • Conocer el entorno ce Flash para poder realizar sus trabajos de una manera rápida y practica
  • Crear interpolaciones en sus diferentes modelos
  • Crear Clip de película para evitar interpolaciones repetitivas y dar una mayor rapidez a su página web
  • Podrá crear sus propios botones con los objetos o imágenes para su página web
  • Podrá crear banners para realizar propaganda en internet

El alumno será capaz de realizar una película la cual tendrá vínculos con sus diferentes escenas la cual podrá utilizar tanto para la web como para exponer temas.

Preguntas frecuentes

¿Con flash puedo realizar una publicidad impresa?
Si se puede realizar pero no es lo mejor ya que existen otros programas cono Photoshop o Illustrator con mayores herramientas para realizar una publicidad impresa.
¿Es cierto que la poner películas de flash en internet se vuelve lenta la página?
Todo depende de que tanta información e interpolaciones cuente tu película ya que mientas mas imágenes y escenas le ingresas a tu presentación se hará más pesado el archivo, lo recomendable es realiza varias películas cortas para ingresarlas en diferentes paginas y no pese tanto el archivo en la web.

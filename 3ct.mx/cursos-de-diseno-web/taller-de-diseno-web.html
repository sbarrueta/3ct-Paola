Taller integral de diseño web

El taller integral de diseño web está contemplado para guiar al alumno por todos los pasos de aprendizaje para lograr un fin de crear un sitio web profesional - desde desarrollar su concepto creativo, hasta subirlo al servidor y darle de alta en los motores de búsqueda.

Este curso de diseño web está dirigido sobre todo a los diseñadores quienes dominan las herramientas básicas de creación de sitios web, tales como por ejemplo Adobe Photoshop, Flash y sobre todo Dreamweaver y quienes desean conocer las técnicas profesionales.

 

Entre semana: 5 semanas 1.5 cada día en los grupos de martes a jueves

Fines de semana: viernes, sábado o domingo, 5 horas, 5 semanas

Hay varios horarios disponibles para el taller de diseño web: por las tardes entre semana y los fines de semana por las mañanas y por las tardes.

 

Los requisitos para tomar el taller es un buen o hasta avanzado dominio de las siguientes temas, tecnologías y aplicaciones:

  • Que son y como funcionan sitios de internet
  • Entendimiento de los códigos XHTML y CSS
  • Buen dominio de Photoshop (puede ser otro editor de mapas de bits, por ejemplo Fireworks o Gimp)
  • Buen dominio de Dreamweaver

En general es suficiente acreditar los cursos de Photoshop y de Dreamweaver intermedio en 3CT para poder directamente entrar al Taller y aprender como diseñar sitios web de alta calidad.

Taller de diseño web: creación profesional de sitios web con la paquetería Adobe

Taller integral de diseño web es una forma de enseñar a los alumnos como crear páginas web profesionales a traves de una metodología paso a paso donde el instructor guía los particpiantes para que puedan obtener los resultados profesionales.

Este taller no contempla enseñar los códigos que forman sitio ni herramientas que sirven para su desarrollo sino más bien el uso correcto de estas para lograr el objetivo. Por esta razón se requiere tener el dominio do las tecnologías mencionadas en la sección de los requisitos para poder participar en este taller.

Otra cosa que distingue el taller de los demás cursos de diseño web es que se trabaja con un proyecto, en otras palabras no es una serie de ejercicios sueltos, sino una secuencia de pasos continuos y consiguientes que permiten desarrollar un proyecto desde cero hasta su presetnacion final en el ambiente web. El proyecto que quiere realizar escoge el alumno a su gusto y puede ser algun proyecto personal o todavia mejor profesional con valor comercial, de esta manera el participante puede recuperar el dinero invertido en la enseñanza combrandose al cliente para quien se realiza el sitio.

 

Objetivos del curso

Los objetivos del taller están enfocados en producir resultados tangibles en forma de tener al fin de las clases un sitio web profesional y plenamente funcional en línea.

Lo más importante aquí es que no se enseña como utilizar las herramientas tipo Dreamweaver sino como usarlas de manera eficiente para diseñar sitios web de alta calidad.

Ejercicios

En el taller no se realiza ejercicios abstractos sino se realiza paso a paso un proyecto real, en otras palabras un sitio web. Este proceso puede ser considerano como una serie de ejercicios cda uno relacionado con otros, tambíen las que están realizadas por otros participantes del taller.

Temario

El temario del taller está organizado según los siguientes pasos:

  • Definición del proyecto
  • Análisis de los recursos
  • Estructura semántica y visual
  • Creación y edición del contenido
  • Desarrolo e implementación
  • Evaluación y mantenimiento
 

Ventajas adquiridas

La principal ventaja de este taller aparte de aprender es poder ganar realizando un proyecto

Cursos que siguen

Después de tomar el taller integral de diseño web se puede estudiar cualquier uno de los talleres del Diplomado Webmaster con Certificación de 3CT o entrar en uno de cursos de programación o bases de datos, por ejemplo:

  • Bases de datos básico
  • Fundamentos de programación

Curso de Visual Basic 6

El curso de Visual Basic 6 trata uno de los lenguajes de programación más populares; su orientación visual lo hace una de las herramientas más amigables para el desarrollo de interfaces gráficas. Como Visual Basic 6 está fuertemente integrado al ambiente Windows, las aplicaciones que se pueden lograr con él se integran perfectamente con este sistema operativo.

El curso de Visual Basic 6 está pensado para un doble perfil. Es excelente para personas que quieren iniciarse en la programación desde un ambiente fácil y cómodo pero que aspiran a programar en Java; es decir, como preparación para entornos más difíciles. Aunque también es el mecanismo natural para aprender a programar en la plataforma .NET, para acceder a la creación de aplicaciones de escritorio y, posteriormente, a la programación web.

Introducción a Visual Basic

Desde su aparición, Visual Basic fue aclamado por todos los programadores consumados de BASIC y aun por los novatos. Fue una oportunidad de migrar las aplicaciones sin interfaz gráfica, o con una muy pobre, al recién llegado sistema operativo Windows, que estaba llamado a liderar el mercado y que iba a servir para establecer el estándar de todas las plataformas para computadoras de propósito general.

Visual Basic tiene un mecanismo de desarrollo intuitivo que separa limpiamente el diseño de las ventanas y los componentes con los que el usuario debe interactuar, del código de programación que hará operativa la aplicación. Por eso, con muy pocas tiempo de trabajo, es posible hacer una solución con todos los componentes comunes de un entorno gráfico: campos de texto, combos, cuadros de diálogo, etc.

En el curso de Visual Basic que ofrecemos en 3CT se explica el funcionamiento del entorno de desarrollo desde lo esencial; es decir, se muestra el uso de los diferentes paneles: el cuadro de herramientas, el explorador de proyectos, la lista de propiedades, etc. Asimismo se estudian los fundamentos de programación, los mecanismos para crear ventanas, la interacción entre ellas y los cuadros de diálogo, etc.

Uno de los beneficios centrales que se obtienen del curso de Visual Basic 6 es que las aplicaciones que se realizan en esa plataforma son casi completamente compatibles con Windows Vista y con Windows 7; así que el conocimiento es inmediatamente aplicable y los programas se podrán implementar casi sin problemas.

Como Visual Basic 6 es un lenguaje de uso muy extendido (de hecho, desde la versión 4, lanzada con Windows 95, todas son de 32 bits y han participado de los avances en la plataforma), existe una gran cantidad de información disponible en español, con múltiples ejemplos, desde la página oficial de MSDN hasta en páginas dedicadas exclusivamente a ese fin.

Con el curso de Visual Basic 6, además, se obtienen los fundamentos de programación necesarios para acceder a la tecnología .NET. Este concepto es la nueva apuesta de Microsoft para competir con los lenguajes dominantes del ambiente de la programación actual. .NET toma muchos de los modelos probados por Java y los integra a Visual Basic, que en las versiones .NET conservada su facilidad para la creación de interfaces gráficas profesionales.

La capacitación en Visual Basic 6 que se ofrece en 3CT está pensada como parte integral de un proyecto que ofrece alternativas de aprendizaje de programación para cualquier necesidad. Si usted quiere formarse como programador comenzando por un lenguaje fácil y amigable, el curso de Visual Basic 6 es su mejor opción.

 

Objetivos del curso

El objetivo central del curso de Visual Basic 6 es formar programadores para crear aplicaciones de escritorio que se ejecutan en todas las versiones vigentes de Windows. Además, también se pretende iniciar en el desarrollo de software a las personas que se interesen por programar en tecnologías más avanzadas como .NET o Java pero que estén buscando una iniciación que les permita formarse con fundamentos más sólidos.

Al terminar el curso de Visual Basic 6, el alumno podrá hacer aplicaciones de escritorio con una interfaz gráfica profesional que se ejecutarán en todas las versiones vigentes del sistema operativo Windows.

Preguntas frecuentes

¿Es verdad que Visual Basic 6 está obsoleto?
No. Aunque desde 2008 Microsoft ha cancelado el soporte para esta versión, el lenguaje sigue siendo muy usado; por otro lado, como las aplicaciones construidas con lo que se aprende en el curso de Visual Basic 6 se pueden ejecutar en las versiones actuales de Windows (como Vista y 7), no hay ninguna razón para no hacer una aplicación en este lenguaje; en especial si va a ser sencilla.
¿Por qué ponen como requisito para el curso de Visual Basic 6 el de .NET?
Como los fundamentos de programación son los mismos y Visual Basic 6 es mucho más fácil, aprender la tecnología resulta más ligero. De hecho, la curva de aprendizaje de Visual Basic 6 es mucho más cómoda que la de la versión .NET. Por otro lado, como esta última ya tiene la orientación a objetos, su compresión se aligera a su vez si ya se tienen nociones de los fundamentos de programación.

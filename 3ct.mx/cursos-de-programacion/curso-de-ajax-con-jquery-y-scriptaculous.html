Curso de JQuery

JQuery es una librería hechas con Javascript, que permiten desarrollar interfaces de usuario web más intuitivas, con mejor interacción, efectos visuales sorprendentes, controles ricos y validaciones de formularios para servicios web, páginas y portales de Internet con ayuda de XHTML, CSS yJQuery.

El curso de JQuery va dirigido a diseñadores o programadores web que busquen mejorar la navegabilidad de sus desarrollos y que cuenten conocimiento en HTML y CSS, asi como conocimiento básico de Javascript.

Utilización de JQuery

Jquery sirve para manipular el DOM de las páginas web a través de interfaces gráficas más amigables e intuitivas para el usuario, que permitan mejorar la experiencia de navegabilidad.

Permite crear efectos como:

  • Carrusel de imágenes 2D y 3D
  • Calendarios visuales
  • Menú dinámico e interactivo.
  • Ordenamiento ascendente y descendente de datos en tablas
  • Arrastrar y soltar elementos de la página
  • Ventanas flotantes
  • Álbum de imágenes

Las librerías y frameworks de JQuery y Scriptaculous son software libre y de código abierto las cuales permiten simplificar la manera de interactuar con los documentos HTML, manipular el árbol DOM, manejar eventos, desarrollar animaciones y agregar interacción a páginas web.

JQuery es la librería web más popular escrita en JavaScript que implementa la tecnología AJAX y que ha permitido revolucionar el concepto de dinamismo en las páginas web a través de poderosas y fáciles funciones que logran efectos visuales sorprendentes y controles ricos (widgets) poderosos que hace 10 años eran imposibles de imaginar dentro de una página web.

Ejemplos de portales que implementan dicha tecnologías son: Google Maps, Facebook, Hotmail, Gmail, Yahoo, Amazon, NASA, CNN, NBC, DELL, Adobe y muchos más.

Preguntas frecuentes

¿Cuál es la diferencia de JQuery y Flash?
JQuery está enfocado a brindar dinamismo a las páginas web, es software libre y código abierto, permite simplificar la manera de interactuar con los documentos XHTML mediante CSS y JavaScript, lo que te permite incorporar efectos visuales ya creados y parametrizados sin necesidad de diseñar la animación desde cero, además incorpora la funcionalidad de envío de formularios, cambiar el contenido de una página web sin necesidad de recargarla; mientras que Flash esta destinado a la producción y entrega de contenido interactivo y multimedia mediante animaciones tipo video y presentaciones ejecutivas; es software de paga, es decir, se debe comprar una licencia y se incrusta dentro de las pagina web, pero no se acostumbra para interactuar ni modificar los contenidos de las paginas HTML.
¿Es cierto que Dreamweaver puede crear elementos de tecnología JQuery como menús desplegables y acordeones?
Sí, aunque para sacar mayor provecho de la tecnología JQuery debemos profundizar en el tema a través del estudio de las librerías JQuery y Scriptaculous que se analizan el curso.
¿Qué editor se puede utilizar para JQuery?
Se puede utilizar desde un bloc de notas hasta un editor profesional como: Dreamweaver, Netbeans, Microsoft Visual Web Developer,Sublime, etc.

Curso de Visual Basic .NET

Visual Basic es uno de los lenguajes que ha alcanzado mayor popularidad a lo largo de la historia de la programación moderna por su amigabilidad para crear interfaces gráficas perfectamente profesionales y funcionales. Con el curso Visual Basic .NET, 3CT ofrece el acceso a una tecnología que ha ganado gran aceptación en el medio del desarrollo de software  y que integra alternativas multiplataforma.

El curso de Visual Basic .NET está dirigido a personas interesadas en la programación visual y orientada a objetos que quieran acceder a las alternativas que esta herramienta obtiene del Framework .NET. Igualmente, el curso de Visual Basic .NET está pensado para quienes ya programan en versiones anteriores de Visual Basic y desean acceder a esta nueva propuesta de Microsoft.

Introducción al curso de Visual Basic .NET en 3CT

El curso de Visual Basic .NET trata de la versión más actual de uno de los lenguajes más importantes de la historia de la programación: BASIC; la idea original de sus creadores fue ofrecer una alternativa que resultara accesible a personas que no tuvieran una formación científica o para enseñar programación a estudiantes universitarios en nivel licenciatura. La aparición de las computadoras personales favoreció el crecimiento de BASIC porque en la inmensa mayoría de ellas fue ofrecido este lenguaje como parte del sistema operativo; de tal suerte que en el final del siglo XX había millones de computadoras caseras con las alternativas de desarrollo de software preinstaladas y disponibles para cualquiera.

Con la aparición de los sistemas operativos con interface gráfica fue necesaria una revolución de BASIC y Microsoft lanzó la versión visual, que ha tenido siempre las herramientas para la creación de aplicaciones con lo necesario para ejecutarse en una ventana del ambiente Windows y que se llamó Visual Basic.

Desde el surgimiento de BASIC han pasado casi 50 años y de la aparición de la versión visual, casi dos décadas; no obstante, la evolución no se ha detenido. A la llegada del siglo XX Microsoft lanza la plataforma .NET cuyas bases asimilan el concepto de máquina virtual introducido por Java y el paradigma de la programación orientada a objetos.

Las versiones de Visual Basic antes de .NET fueron muy criticadas y consideradas por muchos como triviales al compararlas con los lenguajes que ofrecían esas mismas capacidades de programación, como Delphi; la expresión común es que Visual Basic era para programar juguetes. Los fabricantes de Visual Basic .NET se aseguraron de elevar la calidad de esta nueva versión y han logrado un producto muy competitivo que está integrado en multiplataforma al estilo de Java; es decir, se puede programar con Visual Basic igualmente programas stand alone para ejecutarse en todas las versiones vigentes de Windows, con ASP para el servidor IIS en aplicaciones web, así como programas para dispositivos móviles. Entonces, Visual Basic .NET dispones de múltiples alternativas de desarrollo y abre un horizonte de posibilidades de aprendizaje muy amplias.

El curso de Visual Basic .NET que impartimos introduce a los fundamentos del lenguaje, especifica la programación orientada o objetos y da una panorámica de las herramientas para la creación de interfaces gráficas modernas en aplicaciones de escritorio.

Preguntas frecuentes

He escuchado mucho eso de .NET ¿Qué significa?
Es una tecnología desarrollada por Microsoft para atender el avance de las tecnologías de redes que funciona como un Framework, es decir, como un marco de trabajo o entorno común para diferentes lenguajes y sistemas operativos. Funciona de manera semejante a la máquina virtual de Java, de manera que no depende el IDE en que puedan ser desarrolladas las tecnologías. El entorno .NET atiende lo mismo a lenguajes de programación como Visual Basic y C# así como a bases de datos SQL Server y a integraciones web en ASP.

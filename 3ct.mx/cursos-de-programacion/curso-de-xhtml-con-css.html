HTML con CSS: Creación de páginas web

El tema central de HTML, que es un acrónimo para HyperText Markup Language (Lenguaje de marcado de hipertexto), está relacionado con una de las grandes revoluciones tecnológicas de la historia: el concepto de hipervínculo; el uso de este concepto trajo al mundo de la informática un nuevo tipo de textos: aquéllos con los que el usuario puede interactuar haciendo un clic que lo conduce a otros elementos. Esto significa que en un mismo párrafo pueden ser hipervínculos todas las palabras que tengan un significado profundo, útil o trascendente para el usuario, quien puede utilizarlos para dirigirse a diferentes destinos: otro lugar en la misma página, o en la misma aplicación o en un sitio remoto; este último puede referirse a una computadora a pocos metros o a un servidor en otro continente. A esta capacidad de dirigirse a diversos puntos se le conoce hoy como navegación, y sus posibilidades son de alcance mundial debido a lo que hoy se conoce como la Internet. Entonces, el curso de HTML permite que el alumno haga sitios web que tengan hipervínculos internos o a cualquier página web en el mundo.

Sin embargo, aunque las ligas a otros sitios son la capacidad más apreciada de las páginas HTML, las posibilidades del lenguaje van más lejos: se puede manipular la tipografía, el formato y el estilo del texto; mostrar imágenes; ejecutar sonido, música y video; controlar efectos visuales; y crear formularios para que la interacción con el usuario vaya más allá de un simple clic y se pueda enviar información para ser procesada por lenguajes como PHP o JSP y se almacene en bases de datos o en archivos.
Lo dicho hasta aquí muestra los fundamentos del curso de HTML. No obstante, hoy por hoy, los documentos HTML tienen un ámbito más plástico y la tecnología ahora tiene añadidos en lo que se conoce como XHTML: eXtensible HTML. Uno de los añadidos importantes es el lenguaje CSS: Cascading Style Sheets (Hojas de estilo en cascada), que sirve para facilitar la composición de los elementos de las páginas HTML. CSS está pensado para usarse en aplicaciones web sencillas o en sitios grandes cuyo mantenimiento puede ser muy complejo.

En el curso de HTML se darán también los fundamentos de CSS con lo que se aprenderá a trabajar el contenido separado del diseño. HTML controla títulos, párrafos, listas, tablas, imágenes, objetos externos y formularios, mientras que CSS les asigna el espacio que usarán, la tipografía, los colores, márgenes y fondos, etc.

 

Objetivos del curso

Al finalizar el curso de XHTML con CSS, los participantes:

  • Estarán iniciados en la creación de sitios web estáticos
  • Conocerán la estructura básica de un documento HTML
  • Podrán hacer un sitio con hipervínculos hacia el propio documento que los contiene, hacia otros documentos del mismo sitio o hacia documentos externos publicados en cualquier dominio
  • Serán capaces de agregar contenido de texto a una página web y darle formato: títulos, párrafos, listas y tablas
  • Podrán agregarle imágenes a sus páginas web
  • Conocerán los mecanismos HTML para crear formularios que envían datos al servidor
  • Aprenderán la composición con marcos
  • Conocerán las tres maneras de agregar estilo a los objetos HTML con CSS
  • Aprenderán los conceptos de selector, clase y id; así como el de propiedad-valor en el lenguaje CSS
  • Podrán usar los fundamentos de composición de los contenidos de una página web agregando estilo a los divisores con CSS

Ejercicios

Al terminar el curso el alumno habrá realizado una serie de ejercicios que simularán sitios web, por lo que deben:

  • Crear un lanzador de sitios web favoritos
  • Crear un sitio web que sea el curriculum vitae del usuario
  • Crear un sitio web para alta de datos para una empresa de reclutamiento de personal
  • Crear el sitio web de una publicación periódica (por ejemplo una revista)

Preguntas frecuentes

¿HTML es un lenguaje de programación?
No. En realidad es un código para la composición de páginas web.
¿Necesito saber HTML para volverme diseñador web?
En un sentido estricto, sí. Aunque para tomar el curso de Dreamweaver, Photoshop o Flash no se pide como requisito, las tres aplicaciones interactúan con la Internet (Dreamweaver en mucho mayor grado). Al final, en el entorno profesional es indispensable porque sin el conocimiento del código HTML muchas labores resultan penosas o imposibles de comprender.
¿Qué tiene que ver HTML con Java?
Java puede usarse para desarrollar en tres plataformas: para programas de propósito general (como Word), para celulares y otros dispositivos semejantes, y para la Internet. En este último ámbito Java tiene mucha trascendencia y exige un conocimiento puntual de HTML.

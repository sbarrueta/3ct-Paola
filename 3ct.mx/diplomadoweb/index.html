Diplomado web 2.0 «sólo código»

Es un curso intensivo para principiantes que quieran aprender los fundamentos de la programación de sitios web. El objetivo principal del curso es mostrar cada una de las tecnologías involucradas en el desarrollo de aplicaciones para la Internet sin el uso de herramientas de diseño: "sólo código".

Perfil de los aspirantes

Los aspirantes deben haber cursado y aprobado en 3CT, con promedio mínimo de 8.0, los cursos de Fundamentos de programación, Bases de datos I y HTML con CSS; o probar un pleno conocimiento de sus contenidos. Preferentemente, deben tener estudios universitarios.

Estructura del curso

Se verá cómo se programa la capa del cliente con HTLM, CSS y JavaScript; y la del servidor con PHP y MySQL.

Además se mostrarán los fundamentos del uso de AJAX para hacer más eficiente la relación entre ambas capas.

La capa del cliente

Tecnologías de navegador web

Hay tres códigos que todos los navegadores interpretan: HTML, CSS y JavaScript. Estos tres lenguajes se encargan de interactuar con el usuario final, cuya computadora es conocida como 'cliente' porque es la que recibe todas las acciones de los servicios web.

Contenido, diseño e interacción

El código HTML marca los contenidos para determinar el uso que se le da al texto: título, párrafo, lista, tabla, formulario, etc. CSS controla la apariencia de los documentos y puede darles formato, color, estilo y composición. JavaScript puede regular la interacción con el servidor y le da dinamismo a la presentación.

Importancia de las tecnologías 'cliente'

Cuando la Internet surgió, no existían más que lenguajes de servidor. Desde el punto de vista del contenido, los lenguajes cliente son autosuficientes y muchos de los sitios web del mundo sólo los usan a ellos. En tanto que controlan la apariencia y el mensaje, afectan directamente al usuario final; y este es su principal objetivo.

Se tomarán cinco semanas del diplomado para aprender el funcionamiento de JavaScript y la manera como este lenguaje interactúa con los objetos de los documentos HTML; además, se mostrará cómo hay una relación biunívoca entre las propiedades style de dichos objetos en JavaScript y las de los selectores en CSS.

La capa del servidor

Servicios web

Detrás de las páginas que se consultan en la Internet, frecuentemente están funcionando servicios que hacen consultas o procedimientos según las necesidades de los usuarios; estas acciones se realizan con los lenguajes de servidor. Hay varios: ASP, JSP, Perl, etc. El que se usa en la mayoria de los sitios web es PHP.

Consultas y procesos

Cuando los usuarios hacen clic en los botones de los formularios, envían los contenidos de los campos para que los lenguajes como PHP hagan consultas a una base de datos y/o procesen la información por una necesidad específica; cuando se cumple la solución requerida, se crea una respuesta y se envía de vuelta.

Importancia de los servicios web

Para comprender la significiación de los lenguajes de servidor en la Internet actual, basta con saber que sin ellos no existirían ni buscadores ni redes sociales. Pongamos un sólo ejemplo: portales como Facebook trabajan con PHP.

La capa del servidor en el diplomado

Para comprender esta capa se dedicarán siete semanas del diplomado. Durante este tiempo se aprenderán los mecanismos principales de los sevicios web: procesamiento de formularios, conexión a base de datos. creación y control de sesiones y manipulación de cookies. Además, se aprenderá a instalar el servidor Apache y PHP.

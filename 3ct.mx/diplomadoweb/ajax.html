Control entre cliente y servidor con AJAX

Interactividad cliente-servidor

AJAX es un modelo de programación con JavaScript que permite controlar los procesos de solicitud y respuesta entre el cliente y el servidor. Antes de AJAX la actividad del cliente se detenía, luego de cualquier clic, en espera de la respuesta del servidor; ahora, pueden estar en acción simulátneamente.

Redirección de las respuestas del servidor

A la arquitectura del lenguaje JavaScript se le agregó un objeto que puede servir de interfaz entre ambas capas; dicho objeto puede supervisar el tiempo de la respuesta y redirigirla a cualquier etiqueta contenedora del cuerpo del documento que hizo la solicitud; mientras, el usuario puede seguir interactuando con el contenido de dicho documento.

Plasticidad en los documentos HTML

AJAX ha enriquecido la interface gárfica de las aplicaciones web. Se le puede considerar una verdadera revolución para la experiencia del usuario pues significó un aumento de las alternativas de los desarrolladores. Hay nuevos efectos visuales: arrastrar y soltar objetos, autocompletar con listas tomadas de una base de datos, etc.

Introducción a Illustrator

Adobe Illustrator es un programa de diseño vectorial que está destinado para la creación artística del dibujo aunque también se pueden desarrollar diseños para la web. Sus dibujos son creados por formas geométricas que están constituidas por trazos matemáticos a los que se les conoce como vector. Es conocido por la alta calidad en nivel artístico de los diseños creados, podemos realizar ilustraciones tales como dibujos animados, logotipos, personajes, flyers, tarjetas de presentación, diseño para lonas, invitaciones, etc.

Se ha vuelto tan popular que hasta los impresores se han visto en la necesidad de aprender esta herramienta para estar a la vanguardia del diseño vectorial ya que actualmente la mayoría de los diseñadores utilizan Illustrator para realizar todo tipo de publicación y ya casi todo medio impreso tiene que ver con el ambiente digital.

Illustrator salió al mercado en 1987 con un poder innovador y hasta nuestra época ha ido mejorando sus herramientas para desarrollar de una manera más sencilla y profesional los diseños o ilustraciones vectoriales.

Los objetos creados en Illustrator se pueden combinar para crear otros objetos sin perder la nitidez de sus trazos. Se pueden incluir imágenes de mapa de bits como son los formatos .JPG y .GIF entre otros. Con Illustrator también podemos realizar el diseño de páginas web para posteriormente hacerlo en algún programa para web o podemos realizar nuestras páginas web y realizar los hipervínculos en el propio Illustrator.

Como habíamos mencionado, Illustrator es un programa de diseño vectorial, la ventaja de utilizar este tipo de programas es poder ampliar el tamaño de una ilustración sin el temor de sufrir algún deterioro al momento de aplicar el aumento como lo hacen algunos dibujos de formato rasterizado. Actualmente, con Illustrator se pueden crear objetos 3D y esto le da una mayor cobertura en el diseño ya que nos está dando una utilidad sin necesidad de manejar otros programas en 3D.

Illustrator pertenece a la compañía Adobe Systems la cual es dueño de los programas como Photoshop, DreamWeaver, Flash, etc. Illustrator se puede combinar con estos programas; por ejemplo, se pueden diseñar los botones para un efecto Roll-Over y usarlos en DreamWeaver; también se puede crear un logotipo y utilizarlo en Flash sin necesitad de rasterizarlo.

Una de las mejoras desde la versión CS4 es que ya se pueden trabajar con varias páginas en el mismo archivo los que no se podía hacer con las versiones anteriores; ahora en la versión CS5 ya incorporaron le herramienta perspectiva que es de gran utilidad para los trabajos que necesitan utilizar puntos de fuga; otra ventaja es el poder pintar dentro de los objetos seleccionados sin la preocupación de salirse de nuestro objeto así como también pintar por detrás del objeto seleccionado.

Cuenta también con efectos y filtros para darle un mayor realismo a sus ilustraciones y no tener la necesidad de exportar y aplicar dichos efectos en el programa de Photoshop. Ya con estos efectos y el manejo de filtros se obtiene una mayor ventaja al momento de crear los diseños.

El alumno aprenderá a crear dibujos en forma vectorial tanto para el diseño impreso como para la creación de logotipos, botones o diseños para la web; al terminar el curso será capaz de:

Realizar dibujos sencillos en formato vectorial hasta un logotipo profesional

  • Aprenderá a diferenciar y aplicar los modos de color CMYK y RGB
  • Será capaz utilizar las técnicas para desarrollar desde una tarjeta de presentación hasta cualquier tipo de publicación impresa

Ejercicios

Los diseños que realizaran en Illustrator van desde ilustraciones sencillas hasta un diseño profesional para empresas; los ejercicios están diseñados de una manera práctica, divertida y concisa para que puedan desarrollar los diseños de una manera fácil; algunos ejercicios serian los siguientes:

  • Creación de figuras por medio de objetos geométricos
  • Crear un dibujo vectorial utilizando los colores sólidos y degradados
  • Diseño de un empaque sobre medidas especificas y diferentes tipos de líneas
  • Dibujar un personaje animado con la ayuda de la herramienta pluma y las técnicas de combinar y dividir objetos
  • Diseñar el Logotipo y toda la folletería impresa como flyers, tarjetas de presentación, etc., sobre una empresa así como también la propuesta para la web
  • Por medio de las técnicas obtenidas realzar los fondos y botones vanguardistas para una página web

Preguntas frecuentes

¿Para manejar Illustrator tengo que saber dibujar a mano?
No es necesario ya que nosotros te damos las técnicas necesarias para desarrollar tus dibujos desde un dibujo sencillo hasta uno profesional, en este curso básico se enseña desde cómo crear objetos geométricos establecidos por el programa de Illustrator hasta el manejo de curvas que requieren cierta precisión.
¿Dónde puedo entrar a trabajar después de haber tomado el curso Illustrator?
Puedes trabajar como diseñador grafico en cualquier empresa que necesite el puesto correspondiente ya que con este primer curso serás capaz de desarrollar cualquier diseño publicitario tanto para negocios pequeños como empresas grandes. Además puedes tomar otro curso para estar mejor preparado en el ambiente grafico.
¿Se pueden crear páginas web con Illustrator?
Con Illustrator puedes crear la propuesta de cómo quedaría tu pagina web además de crear los botones para darle una mayor presentación a las paginas ya que te permite crear efectos de degradado y realzar el texto de los botones para darle un efecto tipo 3D. También puedes desarrollar las páginas web y realizar hipervínculos con las demás paginas HTML pero en este caso si deseas una página profesional web es mejor que curses a parte del programa de Illustrator otro curso como DreamWeaver para darle una mayor profesionalismo.

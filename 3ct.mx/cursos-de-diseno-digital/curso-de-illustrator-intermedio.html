Curso de Illustrator Intermedio

Illustrator es el programa más utilizado en el ambiente de diseño vectorial ya que con sus herramientas y efectos para la creación de objetos vectoriales ofrece un mayor profesionalismo para la creación de imágenes digitales; este curso está enfocado para utilizar técnicas avanzadas para el desarrollo de efectos para el diseño gráfico y web.

Este curso está dirigido para aquellas personas que ya tomaron el curso de Illustrator básico o que ya tienen conceptos de diseño vectorial como la creación de objetos geométricos, el uso de la herramienta pluma, saber usar puntos de ancla o dominen los temas que se ven en el curso básico. Entonces, este curso está dirigido para aquellas personas que desean ver más a fondo el tratamiento del diseño vectorial.

Illustrator Intermedio – Técnicas avanzadas para el uso del diseño vectorial

En el curso de Illustrator básico el alumno adquirió los conocimientos sobre el entorno de trabajo, activar y desactivar los paneles que desea usar, también aprendió a aplicar transformaciones a los objetos vectoriales como las técnicas de reflejo, escalado, rotar. etc. El alumno obtuvo las técnicas para trabajar con medidas especificas para realizar un dibujo con apoyo de la cuadricula y las líneas guías con la que cuenta Illustrator.

En este curso de Illustrator intermedio el alumno será capaz de desarrollar diseños profesionales con las herramientas y técnicas que nos ofrece el programa vectorial, así como aplicar efectos 3D que es una de las técnicas que se están utilizando actualmente.

Una de la ventajas de illustrator en el tratamiento de texto es la cantidad de opciones que nos ofrece para poder darle una mayor presentación, es decir, nos da las opciones de espaciado entre letras y palabras, uso de interlineado, así como algunos efectos de superíndice, subíndice y versales, también nos permite el uso de las sangrías y los diferentes tipos de alineación.

En la versión CS5, con la integración de a herramienta perspectiva, nos da una mayor facilidad de trabajar con diseños de 1,2 o 3 puntos de fuga de una manera más rápida para poder desarrollar este tipo de publicaciones y con el panel de estilos gráficos se le puede dar un mayor realismo.

Como habíamos mencionado anteriormente Adobe Systems cada vez ha ido creciendo para realizar diseños de una manera mas fácil y divertida al momento de ir desarrollando nuestros diseños tanto para imprenta como para publicaciones web.

Con la integración de varias hojas en un mismo archivo, da una gran flexibilidad al momento de realizar diseños impresos como flyers o trípticos que necesitan estar impresos por ambos lados ya que nos evita estar haciendo un archivo para la parte de enfrente y otro para la parte de atrás.

Illustrator, con el uso del efecto en 3D, le da un mayor alcance básico en el área del diseño industrial ya que permite realizar ilustraciones novedosas y se pueden realizar diseños para stands, como por ejemplo crear una botella e ingresar la etiqueta de la compañía, o simular una lona donde se muestre el logo de una empresa.

Con la herramienta Malla se pueden crear efectos de sombra para aquellos personajes que se diseñen o dibujos que necesiten darle un toque de luz, también nos va a permitir dar un toque de difuminados a dibujos realizados en Illustrator.

Una vez que tengamos los conocimientos necesarios, illustrator nos permite crear nuestras propias plantillas para que no tengamos que volver a realizar todo el trabajo desde cero, sino que con el manejo de plantillas va a ahorrar tiempo tanto para el diseño impreso como para el diseño web.

Con el uso de los símbolos y la herramienta pincel se pueden crear diseños de una manera artística como la creación de paisajes o fondos para la web, y en combinación de los objetos vectoriales le da un toque personalizado para las ilustraciones realizadas.

llustrator es una gran herramienta tanto para el diseño gráfico como para el diseño web ya que con los conocimientos adquiridos se podrá utilizar para ambas cosas, es decir se pueden crear fondos y botones para la web y darle un toque personalizado sin necesidad de utilizar botones o fondos preestablecidos.

 

Objetivos del curso

Adquirir los recursos suficientes para que el alumno sea capaz de crear diseños tanto impresos como para la web y poder tener los conocimientos necesarios para poder trabajar e cualquier empresa de diseño gráfico, ya que con este curso se ven las técnicas avanzadas para creación de diferentes publicidades con el manejo de trazos vectoriales, imágenes y textos.

Ejercicios

  • Desarrollar una publicidad donde apliquen el uso las opciones de texto como el Kerning y Tracking
  • Realizar el diseño artístico utilizando símbolos y pinceles
  • Crea el dibujo de un animal, flor o un súper héroe donde apliquen la técnica de la herramienta malla para trabajar con las tonalidades de color donde simulen efectos de sombra.
  • Con las técnicas adquiridas realizar una propuesta de trabajo para una empresa que necesita todo lo relacionado al diseño impreso y la presentación de su página web.

Preguntas frecuentes

¿Al utilizar el efecto de 3D ya puedo hacer cualquier cosa de diseño industrial?
Illustrator ofrece el efecto 3D de una manera básica como apoyo para tus diseños pero si te necesitas algo más avanzado en 3D tendrías que tomar algún programa enfocada a esta técnica.
¿Puedo realizar una revista con Illustrator?
Si se puede crear una revista pero te llevaría más tiempo estar armando las paginas, es mejor que lo hagas con Indesign que es un programa de diseño editorial. En este caso podrías realizar la portada de tu revista y trabajarla con Indesign.
¿La herramienta Malla funciona igual que el degradado?
Es diferente su combinación de colores ya que con la herramienta de malla te da otro tipo de difuminado en sus colores, además de que puedes limitar el efecto multicolor que te da la herramienta malla.

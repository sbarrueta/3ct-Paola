Curso de Administración de Bases de Datos SQL-Server

La Administración de Bases de Datos es una disciplina que provee las técnicas necesarias para garantizar la disponibilidad, seguridad y eficiente recuperación de la información de una organización.

 

Está dirigido a quienes estén interesados en formarse como profesionales en la administración de las bases de datos,en la programación, ya sea como complemento del desarrollo de software.

Información sobre SQL Server

SQL Server nació en Sybase, posteriormente fue adquirido por Microsof, para competir contra Oracle e IBM.

Sql Server fue orientado hacia los productos de Microsoft de manera gradual, concretandose en la versión 6.0 que fue diseñada para Windows NT

Con el paso de los años SQL Server mejoro en herramientas, desempeño, integración y la minería de datos.

Desde la versión 2005 permite el manejo de XML es decir agrego un tipo de dato XML el cual es convertido internamente a binario antes de almacenarlo.

La ejecución de consultas es cada vez mas eficiente y permite la integración con .NET

Permite la ejecución de vistas para monitorear el estado del sistema, diagnosticar problemas y mejorar el rendimiento.

Actualmente permite el almacenamiento multimedia en las columnas de tipo BLOB, incluyendo también la búsqueda de texto, así como capacidades de en cripta miento de datos.

El manejo de transacciones le permite poder regresar a cualquier momento, en caso de algún fallo en cualquiera de las operaciones involucradas en la transacción.

Existen diferentes versiones tales como:

  • DataCenter
  • Enterprise
  • Standard
  • Web
  • Business Intelligence
  • Workgroup
  • Express

SQL Server permite la interacción de múltiples usuarios de manera concurrente garantizando la integridad en todo momento.

Provee una variedad de servicios tales como:

  • Service Broker
  • Replication Services
  • Transaction replication
  • Merge replication
  • Snapshot replication
  • Analysis Services
  • Reporting Services
  • Notification Services
  • Integration Services

Preguntas frecuentes

¿Puedo administrar una base de datos sin saber Transact?
Las aplicaciones actuales necesitan de una base de datos para funcionar. Sin embargo, en el ámbito laboral o profesional, es difícil que contraten a un administrador que no tenga conocimientos suficientes en programación ya que interactúa frecuentemente con los mismos.
¿Qué diferencia hay entre MySQL y Server?
Ambos sirven exactamente para lo mismo, además, los dos tienen una versión sin costo y una de pago, podríamos decir que la diferencia esta en quien lo usa, es decir Server es usado por las medianas o grandes y My SQL es muy utilizado para los servicios web ya que es uno de los productos que se ofrecen junto con PHP para el Hosting de cualquier pagina web.
¿Cual es la diferencia entre administrar Server y Oracle?
Cada base de datos tiene sus propias herramientas de administración, sin embargo cualquier administrador de base de datos realiza los mismos procesos.
